# 拼豆涂鸦

这是一个基于高德地图的中国地图像素涂鸦单页应用，支持在地图任意位置进行像素绘制。页面采用纯前端静态实现，支持单击与拖动画格、颜色选择、本地自动保存，以及作品导入导出。

## 功能说明

- 使用高德地图作为底图进行展示
- 支持中国地图任意位置像素涂鸦
- 点击右侧 Paint 后进入绘制模式
- 绘制模式下支持单击上色与拖动画格
- 默认绘制颜色为红色
- 支持本地自动保存
- 支持导出作品、导入作品、清空作品
- 缩小时也可以看到已绘制内容，放大后可进行更精细的像素绘制

## 使用说明

1. 使用浏览器打开 wplace.html
2. 等待地图加载完成
3. 点击右侧 Paint 按钮进入绘制模式
4. 在地图上单击或拖动即可进行涂画
5. 如需切换颜色，可在左侧调色板中选择
6. 可通过右侧工具按钮进行导出、导入或清空作品
7. 如页面内容没有更新，请强制刷新浏览器后再试
